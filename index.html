ABOUT THE GAME

This game was developed by Amanda Cuervo Sutil
and it is inspired by Guitar Hero Game.
It was developed as part of my Ironhack Web Development Bootcamp.
The technologies used in this game are HTML5, CSS3 and Javascript ES6.
All the code was done from scratch.
I hope you enjoy playing this game as much as I enjoyed working on it.
Amanda Cuervo - Deciembre, 2022.

HOW TO PLAY

INSTRUCTIONS

Colored squares will start to scroll down from the top.
When they meet their corresponding buttons on the midi
you must press the keyboard button assigned to each square.
Using the music's beat and melody will help you to play.

PRESS A - GREEN
PRESS S - RED
PRESS J - YELLOW
PRESS K - BLUE
PRESS L - ORANGE
PRESS SPACE - SHOOT
